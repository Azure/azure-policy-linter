# Policy Rule Should Contain One Resource Type


| Category | Identifier |
|----------------|----------------------------------------|
| BestPractices | policy-rule-should-contain-one-resource-type |

## Description

The policy rule if is targeting multiple different resource types. Best practice for policy is to target a single resource type at a time, and support multiple resources with initiatives.

### Suggestions

- Split the policy into multiple policies and group them together in an initiative
